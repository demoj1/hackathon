    Варианты заданий

    Первое задание

    Разработать блог.


    Требования:

    ⚀ добавление статей через админ панель;

    ⚁ список статей в блоге (лента);

    ⚂ RSS-рассылка новых статей;

    ⚃ агрегация RSS-рассылок из других источников (например, habrahabr.ru). Периодический сбор рассылки. Сохранение в базу данных статей из источников. Отображение в ленте с возможностью перейти к первоисточнику (“читать далее” -> habr);

    ⚄ (опционально) авторизация через OAuth.

    Второе задание

    Работа с API.


    Требования:

    ⚀ найти публичный API. Примеры: vk, github, twitter;

    ⚁ получить данные для последующей обработки. Тематика данных выбирается самостоятельно;

    ⚂ просмотр обработанных данных и сырых данных. Пример: отобразить все твиты по хештегу, показать среднюю длинну твита, кол-во уникальных авторов по хештегу, самый популярный пост;

    ⚃ (опционально) реализовать операции create, update, delete с данными. API в основном требует авторизацию для подобных операций. Если сможете настроить, будет большой плюс.

    Как сдать задания

    ⚀ выложить проект на открытый git хост (github, bitbucket etc);

    ⚁ задеплоить проект на heroku;

    ⚂ нажать на кнопку сдать проект;

    ⚃ кнопка будет доступна с 6 мая.
